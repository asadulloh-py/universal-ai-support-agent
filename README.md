# 🤖 Universal AI Support Agent

A powerful, low-latency AI Support Agent built with Python, leveraging **Groq AI (LLaMA 3.1)** and integrated seamlessly into **Telegram**. This agent acts as a professional customer support representative for businesses, capable of understanding context, handling complex queries, and responding instantly.

## 🚀 Features
* **Advanced AI Reasoning:** Powered by LLaMA 3.1 via Groq API for lightning-fast inference.
* **Contextual Memory:** Remembers conversation history per user for personalized support.
* **System Prompt Guardrails:** Strictly professional, polite, and business-focused responses.
* **Automated Support:** Handles client FAQs, order status inquiries, and billing assistance 24/7.

## 🛠️ Tech Stack
* Python 3.11+
* `python-telegram-bot` (Telegram API Wrapper)
* `groq` (Groq Cloud SDK)
* `python-dotenv` (Secure environment management)

## 📦 Setup Instructions
1. Clone the repository.
2. Create a `.env` file based on `.env.example` and insert your credentials.
3. Install dependencies and run `python agent.py`.
